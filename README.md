# 数据结构与算法 - Python 学习指南

本项目包含常见数据结构与算法的 Python 实现示例，适合学习参考。

## 文件结构

| 文件 | 内容 |
|------|------|
| `01_stack.py` | 栈的应用：括号匹配、后缀表达式、单调栈 |
| `02_queue.py` | 队列应用：约瑟夫问题、滑动窗口、生产者消费者 |
| `03_linked_list.py` | 链表：LRU缓存、合并链表、反转链表 |
| `04_hash_table.py` | 哈希表：两数之和、字母异位词、最小覆盖子串 |
| `05_binary_tree.py` | 二叉树：遍历、BST、验证、LCA |
| `06_heap.py` | 堆：Top K、合并有序列表、中位数、哈夫曼编码 |
| `07_graph.py` | 图：BFS/DFS、Dijkstra、拓扑排序、Prim |
| `08_sorting.py` | 排序算法：冒泡、快排、归并、堆排、计数、桶排 |
| `09_searching.py` | 查找算法：二分、插值、斐波那契、旋转数组 |
| `10_dp_recursion.py` | 动态规划：斐波那契、背包、LIS、编辑距离 |
| `11_backtracking.py` | 回溯算法：全排列、N皇后、组合、子集 |

## 运行方式

```bash
python 01_stack.py
python 02_queue.py
# 以此类推...
```

## 学习建议

1. **理解核心思想** - 每个数据结构都有自己的适用场景
2. **分析复杂度** - 时间复杂度和空间复杂度是评价算法的关键
3. **多做练习** - 纸上得来终觉浅，实践出真知
4. **总结归纳** - 类似问题往往有相似的解题思路

## 复杂度速查表

### 数据结构操作复杂度

| 数据结构 | 访问 | 搜索 | 插入 | 删除 |
|----------|------|------|------|------|
| 数组 | O(1) | O(n) | O(n) | O(n) |
| 链表 | O(n) | O(n) | O(1) | O(1) |
| 哈希表 | - | O(1) | O(1) | O(1) |
| 二叉搜索树 | O(log n) | O(log n) | O(log n) | O(log n) |
| 堆 | O(1) | O(log n) | O(log n) | O(log n) |

### 排序算法复杂度

| 算法 | 平均 | 最坏 | 空间 |
|------|------|------|------|
| 冒泡排序 | O(n²) | O(n²) | O(1) |
| 快速排序 | O(n log n) | O(n²) | O(log n) |
| 归并排序 | O(n log n) | O(n log n) | O(n) |
| 堆排序 | O(n log n) | O(n log n) | O(1) |
| 计数排序 | O(n+k) | O(n+k) | O(k) |
| 桶排序 | O(n+k) | O(n²) | O(n+k) |
